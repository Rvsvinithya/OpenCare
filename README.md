# OpenCare
##  Overview
 
OpenCare is a multi-modal AI-driven healthcare platform designed to assist doctors in diagnosing critical medical conditions and obtaining AI-powered second opinions.  
The system integrates specialized machine learning models, a real-time medical chatbot (Llama-3), and a notification system to enhance diagnostic accuracy and collaboration among healthcare professionals.

Designed and implemented a real-time notification system to alert doctors for urgent cases and specialist consultations

 
##  Problem Statement
 
Misdiagnosis and delays in diagnosis due to limited access to specialized healthcare professionals can severely impact patient outcomes.  
Traditional healthcare systems rely heavily on manual expertise, where AI remains underutilized.
 
OpenCare aims to:
- Improve diagnostic accuracy  
- Reduce human errors  
- Enable AI-assisted second opinions  
- Enhance collaboration between healthcare professionals
